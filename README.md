## ğŸ› Troubleshooting

### **Preview not updating**
- Click "Show Preview" to enable live preview
- Make sure you've filled in required fields (name, at least one section)
- Try toggling preview off and on again

### **PDF download not working**
- Ensure all required fields are filled
- Check browser console for errors (F12)
- Try a different browser (Chrome recommended)

### **Photo not showing in PDF**
- Make sure photo file is JPG or PNG format
- Keep photo file size under 5MB
- Try uploading a different photo

### **Text overlapping in PDF**
- This is usually fixed automatically
- Try switching to a different template
- Reduce text length in affected sections

### **Resume Score not showing**
- Fill in more sections (minimum: Personal Info, Education, Experience)
- Click "Show Resume Score" button
- Wait a few seconds for analysis to complete
