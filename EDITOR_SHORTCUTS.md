# Rich Text Editor Shortcuts & Tips

Quick reference guide for using the smart rich text editor in the Resume Builder.

## Smart Enter Key Behavior

### üéØ Bullet Points

**Starting a bullet list:**
1. Click the bullet button (‚ò∞) in toolbar, OR
2. Type `‚Ä¢ ` at the start of a line

**Continuing bullets:**
- Type your first bullet point
- Press **Enter** ‚Üí Next bullet automatically appears

**Exiting bullet mode:**
- Press **Enter** on an empty bullet ‚Üí Removes empty bullet and exits list mode

**Example:**

### üî¢ Numbered Lists

**Starting a numbered list:**
1. Click the numbered list button (‚â°) in toolbar, OR
2. Type `1. ` at the start of a line

**Continuing numbered lists:**
- Type your first numbered item
- Press **Enter** ‚Üí Next number (2.) automatically appears
- Type your second item
- Press **Enter** ‚Üí Next number (3.) automatically appears
- Numbers auto-increment!

**Exiting numbered mode:**
- Press **Enter** on an empty number ‚Üí Removes empty number and exits list mode

**Example:**
```

2. Second item [Press Enter - number auto-increments!]
3. Third item [Press Enter]
4. [Press Enter on empty number - exits list mode]
```

### üìù Line Breaks (Without Continuing List)

**Need a line break inside a bullet/number?**
- Press **Shift+Enter** ‚Üí Creates a simple line break
- Does NOT create a new bullet or number
- Useful for multi-line descriptions

**Example:**
```

  And this is a continuation of the same point [Used Shift+Enter]
  Still the same bullet point [Used Shift+Enter again]

```

## Formatting Shortcuts

### Bold Text
**Method 1:** Select text ‚Üí Click **B** button
**Method 2:** Type `**text**` manually

**Example:**
```

```

### Italic Text
**Method 1:** Select text ‚Üí Click *I* button
**Method 2:** Type `*text*` manually

**Example:**
```
This is *italic text* in a sentence
```

## Quick Tips

### ‚úÖ DO:

- Use **bold** for emphasis on achievements

## Troubleshooting

**Q: How do I stop the auto-bullets?**
A: Press Enter on an empty bullet point

**Q: How do I add a line break without creating a new bullet?**
A: Press Shift+Enter instead of Enter

**Q: The numbers aren't incrementing correctly**
A: The editor counts all numbered items in the text. Start fresh or manually adjust.

**Q: Can I use dashes (-) or asterisks (*) for bullets?**
A: Yes! Type `- ` or `* ` and they'll work like `‚Ä¢ `

**Q: How do I make text both bold AND italic?**
A: Type `***text***` or wrap with both: `**_text_**`
